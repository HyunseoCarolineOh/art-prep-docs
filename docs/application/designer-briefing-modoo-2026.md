## 📋 프로젝트 개요

### 아트프렙(ArtPrep)이란?
**비전 AI 기반 미대입시 실기자료 분석·큐레이션 플랫폼**

- **핵심 가치**: 6년 도메인 채점 기준 + Vision AI + 합격 DB 1만건으로 수험생의 그림을 10초 내 5단계 분석
- **타겟**: 연 5만명 미대 수험생 (특히 비수도권·정보 격차층)
- **문제 해결**: 일평균 132분 탐색 시간 → 10초 단축, 판단 오류율 65% → AI 기반 객관적 추천
- **차별화**: 3축 모트 (프롬프트 트리 + DB 라벨링 + 행동 메트릭) = 복제 불가 진입장벽

### 이미지 5장의 목적
심사위원(주로 창업·투자 비전문가)에게 **10초 내 핵심 전달**:
1. 문제가 명확하고 크다 (시장 니즈)
2. 솔루션이 혁신적이다 (기술 차별화)
3. 실행 가능성이 높다 (트랙션 + 팀)
