## 0. 영상 제작 전략

### 0-1. 왜 2가지 안인가?
- **A안 (직접 촬영)**: 대표의 진정성·현장 경험을 직접 전달 → 심사위원 신뢰 구축 (감정 전달 우선)
- **B안 (슬라이드)**: 데이터·비주얼 중심 → 촬영 리소스 부족 시 대안 (정보 전달 우선)
- 최종 선택: 촬영 여건·시간에 따라 CEO 결정 (둘 다 준비해두면 리스크 분산)

### 0-2. 공통 원칙
- **첫 3초 훅**: "132분 → 10초" 통계로 즉시 관심 유발
- **자막 필수**: 심사위원이 음소거 환경에서도 이해 가능
- **엔딩 CTA**: "모두의 창업으로 전국 정보 격차 해소" 명확한 Ask
- **길이**: 30~60초 엄수 (편집 시 여유 5초 남기기 → 55초 목표)

## A안: 직접 촬영 버전 (30~60초)

### A-1. 타겟
- 심사위원에게 대표의 진정성 + 문제 이해도 + 실행 역량 직접 전달
- 도메인 경험 6년 + 트랙션 검증 완료 → "이 팀은 실행할 수 있다" 신뢰 구축

### A-2. 스크립트

---

#### [0~10초] — 훅 + 자기소개

**화면**: 대표 정면 샷 (상반신, 배경 깔끔한 단색 또는 작업 공간)

**대사**:
```
안녕하세요, 아트프렙 대표 오현서입니다.
저는 6년간 200명의 미대입시생을 컨설팅하며 구조적인 문제를 목격했습니다.
```

**자막**:
- 0~3초: "안녕하세요, 아트프렙 대표 오현서입니다."
- 3~10초: "6년간 200명 미대입시생 컨설팅 → 구조적 문제 목격"

**촬영 TIP**:
- 조명: 링라이트 또는 자연광 (얼굴 명확히)
- 배경: 화이트 또는 그레이 벽, 또는 작업 공간 (책상·모니터 등)
- 카메라: 스마트폰 고정 촬영 (삼각대 필수, 흔들림 금지)
- 복장: 비즈니스 캐주얼 (셔츠 또는 단정한 티셔츠)

---

#### [10~20초] — 문제 제시

**화면 전환**: 문제 통계 그래픽 (전체화면)

**그래픽 내용**:
- **132분** (큰 숫자, 예: 120pt 굵게)
- "일평균 SNS 탐색 시간" (작은 텍스트)
- **65%** (큰 숫자, 빨간색)
- "판단 오류율 (취향 기반 선택)" (작은 텍스트)

**대사** (보이스오버):
```
연 5만명 수험생이 일평균 132분을 SNS 탐색에 허비하고,
85%가 입시 기준이 아닌 취향으로 자료를 선택해 판단 오류율이 65%에 달합니다.
```

**자막**:
- 10~15초: "132분 SNS 탐색 허비 / 판단 오류율 65%"
- 15~20초: "85% 취향 기반 선택 → 방향 정체 반복"

**촬영 TIP**:
- 그래픽은 이미지 파일로 미리 제작 (이미지 브리핑 활용)
- 보이스오버 녹음 후 그래픽 위에 오버레이

---

#### [20~35초] — 솔루션 제시

**화면 전환**: 모바일 앱 UI 시연 (스크린 녹화 또는 목업)

**UI 시연 내용**:
1. 그림 업로드 버튼 탭
2. AI 분석 진행 (로딩 애니메이션)
3. 5단계 평가 결과 표시 (막대 그래프 또는 별점)
4. 큐레이션 결과 (합격작 Top10 그리드)

**대사** (보이스오버):
```
아트프렙은 비전 AI와 합격 DB 1만건 교차분석으로
이 132분을 10초로 단축하고, 합격 패턴 기반 큐레이션을 제공합니다.
```

**자막**:
- 20~27초: "비전 AI + 합격 DB 1만건 교차분석"
- 27~35초: "132분 → 10초 단축 / 합격 패턴 큐레이션"

**촬영 TIP**:
- UI 프로토타입 없으면 와이어프레임 목업으로 대체 (Figma 화면 녹화)
- 실제 앱처럼 보이도록 스마트폰 프레임 오버레이

#### [35~50초] — 트랙션 제시

**화면 전환**: 트랙션 통계 그래픽 (전체화면)

125명의 사전신청을 확보하며 시장 니즈를 검증했습니다.

**자막**:
- 35~43초: "MVP 전환율 22% (업계 평균 2.2배)"

### B-3. 슬라이드별 비주얼 디테일

#### 슬라이드 1 (0~8초) — 문제 통계
- **배경**: 다크 네이비 그라데이션
- **좌측**: **132분** (120pt, 화이트)
- **우측**: **65%** (120pt, 레드)
- **하단**: "입시 기준 아닌 취향으로 선택 → 방향 정체 반복" (24pt)
- **애니메이션**: 숫자 카운트업 효과 (0 → 132, 0 → 65)

#### 슬라이드 2 (8~16초) — AI 솔루션
- **배경**: 화이트
- **중앙**: 5단계 플로우 차트 (아이콘 + 화살표)
  1. 업로드 → 2. AI 분석 → 3. 채점 매칭 → 4. DB 교차분석 → 5. 큐레이션
- **하단**: "132분 → 10초 단축" (36pt, 액센트 컬러)
- **애니메이션**: 화살표 순차 등장 (좌→우)

#### 슬라이드 3 (16~24초) — 3축 모트
- **배경**: 라이트 그레이
- **중앙**: 3개 열 레이아웃
  - 좌: 트리 아이콘 + "6년 도메인 채점 기준 코드화"
  - 중: DB 아이콘 + "합격 DB 1만건 라벨링"
  - 우: 그래프 아이콘 + "사용자 행동 편차 메트릭"
- **하단**: "복제 불가 진입장벽 1~2년" (24pt)
- **애니메이션**: 3축 동시 페이드 인

#### 슬라이드 4 (24~36초) — 트랙션
- **배경**: 화이트
- **레이아웃**: 가로 타임라인 (좌→우 화살표)
  - 2023: 트로피 아이콘 + "경진대회 1위"
  - 2025: 설문 아이콘 + "150명 설문 검증"
  - 2026.01: 모바일 아이콘 + "MVP 전환율 22%"
- **강조 박스**: "사전신청 125명 확보" (액센트 컬러 배경)
- **애니메이션**: 타임라인 순차 등장 (좌→우)

#### 슬라이드 5 (36~48초) — 팀 + 로드맵
- **배경**: 네이비 그라데이션
- **좌측 (40%)**: 팀 프로필 3명 (실루엣 아이콘 + 이름 + 1줄 설명)
- **우측 (60%)**: 로드맵 타임라인 (세로 배치)
  - 2026 상반기: "MVP 출시 + 파일럿 100명"
  - 2026 하반기: "MAU 1,000명 + B2B 5개"
  - 2027: "시드 1억원 유치"
- **애니메이션**: 팀·로드맵 동시 페이드 인

#### 슬라이드 6 (48~60초) — 엔딩
- **배경**: 풀스크린 액센트 컬러 (오렌지 또는 블루)
- **중앙**: 로고 (있으면) + "ArtPrep"
- **하단**: "6년의 채점 경험이 AI에 들어갑니다" (36pt, 화이트)
- **애니메이션**: 로고 페이드 인 + 텍스트 슬라이드 업
