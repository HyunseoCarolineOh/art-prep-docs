1. 서명/날인 자리에 커서 위치
2. **입력 → 그림** (단축키: Ctrl+N, I)
3. `C:\Users\ohsca\OneDrive\문서\1. archive\각종증명서\서명.png` 선택
4. 크기 조정 (보통 30~40mm 폭, 또는 표 셀 높이에 맞춤)
5. **본문과의 배치**를 "글 앞으로" 또는 "글 뒤로"로 설정해서 위치 미세 조정

1. 4개 양식 모두 작성 + 서명 삽입 후 **PDF로 내보내기** 또는 **.hwp 그대로 저장**

3. 아래 파일들 함께 업로드:

   - 국세 완납증명서 (홈택스)
   - 지방세 완납증명서 (위택스)
   - IR 발표자료 (제작 중)
