📋 필수 항목 2 — 아이디어를 떠올린 배경 이야기
========================================================

저는 2년간 200명 이상의 미대입시생을 컨설팅하며 구조적인 문제를 직접 목격했습니다. 학생 85%가 입시 평가 기준이 아닌 개인 취향으로 실기 자료를 선택해 방향 정체와 재설정 사이클을 반복했고, 이는 개별 학생의 문제가 아니라 올바른 판단을 도울 도구 자체가 부재했기 때문입니다.

2023년 홍익대 경영대학 아이디어 경진대회 1위 수상을 계기로 문제를 본격적으로 검증하기 시작했고, 2025년 150명 온라인 설문과 10명 심층 인터뷰에서 78%가 탐색 비효율과 분석 부재를 최우선 과제로 지목, 67%가 월 3만원 이상 유료 결제 의향을 보였습니다.

결정적 계기는 2024년 비전 AI 모델의 실용화였습니다. 이미지 기반 실기 분석이 처음으로 상업적 실현이 가능해졌고, 2년간 체득한 채점 기준을 코드화한 분기형 프롬프트 트리와 합격 DB를 결합하면 신생팀이 복제 불가한 데이터 모트를 구축할 수 있다는 확신이 들었습니다.

2026년 1월 출시한 MVP 랜딩페이지는 전환율 22%를 달성하며 업계 평균(5~10%) 대비 2.2배 성과를 입증했고, 현재까지 125명의 사전신청을 확보하며 시장 검증 단계를 완료했습니다. 사전신청자들과의 1:1 인터뷰에서 "드디어 이런 서비스가 나왔다"는 반응을 확인했고, 이제는 본격적인 서비스 개발과 데이터 구축으로 전국의 정보 격차를 해소할 시점입니다.

📋 필수 항목 3 — 누구의 어떤 문제를 해결해주나요?
========================================================

【대상】연 5만명 미대 수험생 — 특히 비수도권 수험생과 학종·비실기 전형 준비생

【문제 1 — 시간 낭비】
일평균 132분을 비정형 SNS(인스타그램·핀터레스트·미대입시닷컴) 탐색에 허비합니다. 입시 특화 검색 도구가 없어 키워드를 바꿔가며 수백 장을 넘기지만, 내 목표 학교·전형에 맞는 자료인지 판단할 기준이 없습니다.

【문제 2 — 판단 오류】
학생 85%가 입시 평가 기준이 아닌 개인 취향으로 자료를 선택하여 판단 오류율 65%에 달합니다. "이 그림이 예쁘다"와 "이 그림이 합격 패턴에 부합한다"는 완전히 다른 기준인데, 이를 구분할 도구가 시장에 전무합니다.

【문제 3 — 정보 격차】
합격 데이터와 컨설팅은 수도권 거점 학원에 집중되어, 지방 수험생은 정보 접근 차단 + 이동·거주비 이중 부담을 떠안습니다. 학원비는 월 200만원에 달해 경제적 진입장벽도 높습니다.

【문제 4 — 분석 도구 부재】
현행 플랫폼(미대입시닷컴·인스타·핀터레스트)은 커뮤니티·취향 기반 검색에 머물러, AI 분석 + 합격 DB 기반 전략형 학습 도구가 시장에 전무합니다.

📋 필수 항목 4 — 아이디어를 어떻게 실현하고 싶으신가요?
========================================================

【Phase 1 (2026 상반기) — MVP 개발 및 파일럿 검증】
• 핵심 기능 3종 개발: ①비전 AI 기반 5단계 실기 분석 리포트, ②합격 DB 1만건 교차분석 큐레이션, ③사용자 행동 데이터 수집 파이프라인
• 로컬 프로토타입 완성 → 모바일 최적화 웹앱(Next.js, Supabase) MVP 정식 출시
• 사전신청자 125명 중 파일럿 사용자 100명 확보, 반복 사용률 40% 이상 검증
• 팀원 네트워크 학원 및 입시생 커뮤니티 통해 베타 테스트 진행

【Phase 2 (2026 하반기) — 데이터 고도화 및 B2C 본격화】
• 데이터 라벨링 담당자 채용, 합격 DB 2,000건 → 5,000건 확대
• 데이터분석가 채용, AI 분석 정확도 고도화 (대학·전형별 오차 95% 이상 정확도 목표)
• 행동 데이터 기반 개인화 알고리즘 1차 고도화 (취향 vs 합격기준 편차 메트릭 적용)
• MAU 1,000명 달성, 유료전환율 25% 검증
• B2B 제휴 학원 5개 확보, 인사이트 리포트 제공 시작

【Phase 3 (2027년) — B2B 본격화 및 플랫폼 확장】
• 합격 DB 1만건 달성, 데이터 품질 표준화 체계 구축
• B2B SaaS 라인 정식 출시 (학원용 학생 관리 대시보드 + 입시 트렌드 인사이트 리포트)
• 제휴 학원 10개 이상 확대, B2B 매출 본격화
• 시드 투자유치 1억원 (MAU 1,000명 안정화 근거)
• 특허 3건 출원 완료 (AI 분석 알고리즘, 상표, UI/UX 디자인)

【기술 구현 방식】
• AI 분석 엔진: 비전 AI API + 2년 채점 기준 분기형 프롬프트 트리 (도메인 지식 코드화)
• 데이터베이스: Supabase PostgreSQL + Storage (합격작 정형화 파이프라인 자동화)
• 웹앱: Next.js 반응형 웹앱, 모바일 first UX 설계
• 행동 데이터 파이프라인: Supabase Realtime 이벤트 수집 → Rule-based 가중치 조정 → ML 기반 보정 고도화

【수익 모델】
• B2C: 프리미엄 구독 월 29,900원 (무제한 AI 분석 + 개인화 큐레이션)
• B2B: 학원용 인사이트 리포트 월 199,000원 (학생 데이터 기반 입시 트렌드 분석)
• LTV 538,200원 / CAC 목표 10만원 이하 → LTV/CAC 비율 5.4 (SaaS 건강 기준 3.0 상회)
