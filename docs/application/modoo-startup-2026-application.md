### 1-2. 아이디어를 떠올린 배경 이야기를 들려주세요

```
저는 6년간 200명 이상의 미대입시생을 컨설팅하며 구조적인 문제를 직접 목격했습니다. 학생 85%가 입시 평가 기준이 아닌 개인 취향으로 실기 자료를 선택해 방향 정체와 재설정 사이클을 반복했고, 이는 개별 학생의 문제가 아니라 올바른 판단을 도울 도구 자체가 부재했기 때문입니다.

2023년 홍익대 경영대학 아이디어 경진대회 1위 수상을 계기로 문제를 본격적으로 검증하기 시작했고, 2025년 150명 온라인 설문과 10명 심층 인터뷰에서 78%가 탐색 비효율과 분석 부재를 최우선 과제로 지목, 67%가 월 3만원 이상 유료 결제 의향을 보였습니다.

결정적 계기는 2024년 비전 AI 모델의 실용화였습니다. 이미지 기반 실기 분석이 처음으로 상업적 실현이 가능해졌고, 6년간 체득한 채점 기준을 코드화한 분기형 프롬프트 트리와 합격 DB를 결합하면 신생팀이 복제 불가한 데이터 모트를 구축할 수 있다는 확신이 들었습니다.

2026년 1월 출시한 MVP 랜딩페이지는 전환율 22%를 달성하며 업계 평균(5~10%) 대비 2.2배 성과를 입증했고, 현재까지 125명의 사전신청을 확보하며 시장 검증 단계를 완료했습니다. 사전신청자들과의 1:1 인터뷰에서 "드디어 이런 서비스가 나왔다"는 반응을 확인했고, 이제는 본격적인 서비스 개발과 데이터 구축으로 전국의 정보 격차를 해소할 시점입니다.
```

### 1-3. 누구의 어떤 문제를 해결해주나요?

```
【대상】연 5만명 미대 수험생 — 특히 비수도권 수험생과 학종·비실기 전형 준비생

【문제 1 — 시간 낭비】
일평균 132분을 비정형 SNS(인스타그램·핀터레스트·미대입시닷컴) 탐색에 허비합니다. 입시 특화 검색 도구가 없어 키워드를 바꿔가며 수백 장을 넘기지만, 내 목표 학교·전형에 맞는 자료인지 판단할 기준이 없습니다.

【문제 2 — 판단 오류】
학생 85%가 입시 평가 기준이 아닌 개인 취향으로 자료를 선택하여 판단 오류율 65%에 달합니다. "이 그림이 예쁘다"와 "이 그림이 합격 패턴에 부합한다"는 완전히 다른 기준인데, 이를 구분할 도구가 시장에 전무합니다.

【문제 3 — 정보 격차】
합격 데이터와 컨설팅은 수도권 거점 학원에 집중되어, 지방 수험생은 정보 접근 차단 + 이동·거주비 이중 부담을 떠안습니다. 학원비는 월 200만원에 달해 경제적 진입장벽도 높습니다.

【문제 4 — 분석 도구 부재】
현행 플랫폼(미대입시닷컴·인스타·핀터레스트)은 커뮤니티·취향 기반 검색에 머물러, AI 분석 + 합격 DB 기반 전략형 학습 도구가 시장에 전무합니다.
```

### 이미지 2 — AI 솔루션 (Solution)

**제목**: 비전 AI 분석 + 합격 DB 1만건 교차분석 → 10초 큐레이션

**비주얼 구성**:

**핵심 메시지**: 132분 → 10초 단축, 합격 기준 기반 객관적 추천

## 🎯 6. 리뷰 포인트 (심사위원 시각)

### 고득점 요소
1. **고객 니즈 명확성**: 132분 탐색 시간, 65% 판단 오류율 — 정량적 문제 제시
2. **창업자 전문성**: 6년 현장 경험 + 합격생 50명 배출 — 도메인 신뢰
3. **구현 가능성**: 로컬 프로토타입 완료 + MVP 랜딩 22% 전환 — 실행력 입증
4. **수익 모델 논리**: LTV/CAC 5.4 + B2C·B2B 이중 채널 — 지속가능성
5. **차별화 모트**: 3축(프롬프트 트리·DB 라벨링·행동 메트릭) 복제 불가 — 기술 진입장벽

### 예상 질문 대응
**Q1. 비전 AI 모델 선정은 완료되었나요?**
→ A. 현재 검토 단계이며, 모두의 창업 선정 시 기술 멘토링을 통해 최종 결정 예정입니다. 프롬프트 트리 구조는 모델 독립적으로 설계되어 있어 확장성을 확보했습니다.

**Q2. 합격 DB 1만건 확보 방법은?**
→ A. ①대표 및 팀원 네트워크 학원 20개 협의 중 (5년간 구축한 도메인 신뢰 기반), ②합격생 직접 컨택(팀원 네트워크 200명+), ③온라인 커뮤니티 제휴 (미대입시닷컴, 인스타그램 입시 계정), ④데이터 라벨링 담당자 채용(2026 상반기)으로 단계적 구축 예정입니다.

**Q3. 기존 미대입시닷컴과의 차별점은?**
→ A. 미대입시닷컴은 게시판 커뮤니티로 분석·큐레이션 기능이 전무합니다. 아트프렙은 ①AI 5단계 분석 리포트, ②합격 DB 교차분석 큐레이션, ③행동 데이터 개인화로 입시 전략 학습 도구를 제공하는 완전히 다른 카테고리입니다.

**Q4. 지방 수험생 접근성 개선 구체적 방안은?**
→ A. 클라우드 기반 모바일 웹앱으로 전국 동등 접근 보장, 월 29,900원 구독료로 학원비(월 200만원) 대비 90% 이상 비용 절감, 비대면 큐레이션으로 수도권 이동·거주비 부담 제거가 핵심입니다.
