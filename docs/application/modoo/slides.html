SLIDE 3/5

솔루션 — 3축 모트

신생팀이 복제 불가능한 3축 결합

1. 팀 12년 도메인 채점 기준의 코드화

분기형 프롬프트 트리 — 입시 채점 항목(구도·명암·소재·완성도·표현력)을 분기 노드로 코드화

2. 합격 DB 라벨링 체계

합격작 1만건 + 입시 기준 라벨 — 각 노드에서 합격 DB 라벨과 비교 → 5단계 평가 + 개선 포인트 산출

3. 사용자 행동 편차 메트릭

저장·열람·필터 vs 합격 기준의 편차 — 사용자 행동 데이터가 시간이 갈수록 모델 fine-tuning 신호로 누적

  • 3축이 매칭 알고리즘에 직접 학습 신호로 들어감
  • 사용 누적 시 개인화 정확도 95%+ 수렴
  • 데이터·시간 비대칭이 1~2년 진입장벽 형성
