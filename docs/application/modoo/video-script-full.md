# 영상 2종 상세 스크립트 — 모두의 창업 2026

> **프로젝트**: 아트프렙(ArtPrep) 모두의 창업 지원서 제출용 영상  
> **길이**: 30~60초 (최대 60초 엄수)  
> **용도**: 지원서 첨부 영상 (심사위원 설득용)  
> **촬영·편집 일정**: 2026년 5월 13일  
> **작성일**: 2026-05-11

---

## 📋 공통 가이드

### 목표
심사위원에게 **60초 내 핵심 전달**:
1. 문제가 명확하고 크다 (시장 니즈)
2. 솔루션이 혁신적이다 (AI + 데이터)
3. 실행 가능성이 높다 (트랙션 + 팀)
4. **감정적 공감** (창업자의 진정성)

### 핵심 메시지
**"6년의 채점 경험이 AI에 들어갑니다. 전국의 정보 격차를 데이터로 해소하겠습니다."**

### 기술 사양
- **길이**: 30~60초 (60초 초과 시 탈락 위험)
- **포맷**: MP4 (H.264 코덱 권장)
- **해상도**: 최소 1080p (1920×1080)
- **비율**: 16:9 (가로형) 또는 9:16 (세로형, 모바일 최적화)
- **자막**: 필수 (모든 대사 자막 처리, 음소거 환경 대비)
- **파일명**: `modoo-artprep-video.mp4`

### 구성 원칙
- **첫 3초가 승부**: 통계 수치로 훅 (132분 → 10초)
- **중간 30초**: 솔루션 + 차별화 (AI·DB·모트)
- **마지막 10초**: Ask + 감정 호소 (전국 정보 격차 해소)
- **음악**: 저작권 없는 배경음악 (밝고 희망적인 톤, 볼륨 -20dB 이하)

## 🎬 A안: 직접 촬영 버전 (추천)

### 개요
- **타겟**: 심사위원에게 대표의 진정성 + 문제 이해도 직접 전달
- **강점**: 창업자의 얼굴·목소리 = 신뢰감·진정성 극대화
- **약점**: 촬영·편집 리소스 필요, 대표 촬영 부담

---

### 상세 스크립트 & 타임라인

#### [0~10초] 오프닝 — 문제 제기

**화면**
- 대표 오현서 정면 샷 (허리 위 프레임)
- 배경: 깔끔한 단색 배경 (화이트 또는 라이트 그레이)
- 조명: 자연광 또는 링라이트 (얼굴 명확히, 그림자 최소화)

**대사**
```
안녕하세요, 아트프렙 대표 오현서입니다.
저는 6년간 200명의 미대입시생을 컨설팅하며
구조적인 문제를 목격했습니다.
```

**자막** (화면 하단 1/3 위치)
```
오현서 | 아트프렙 대표
6년 미대입시 컨설팅 경험
```

**연출 포인트**
- 표정: 진지하되 따뜻함 (과도한 미소 지양)
- 말투: 또박또박, 자신감 (너무 빠르지 않게)
- 시선: 카메라 정면 응시

#### [10~22초] 문제 심화 — 통계로 구체화

**화면 전환**
- 컷 전환 (Cut) 또는 페이드 (Fade) → 통계 그래픽 화면
- 전체 화면을 통계 그래픽으로 채움 (대표 화면 종료)

**그래픽 내용** (After Effects 또는 Canva 제작)
```
┌─────────────────────────────────┐
│  일평균 132분 탐색 허비 ⏱       │
│  (큰 숫자: 132분, Amber-500)    │
│                                 │
│  판단 오류율 65% ❌             │
│  (큰 숫자: 65%, Red-500)        │
│                                 │
│  → 입시 기준 아닌 취향으로 선택 │
└─────────────────────────────────┘
```

**내레이션** (대표 보이스오버 또는 현장 녹음)
```
연 5만명 수험생이 일평균 132분을 SNS 탐색에 허비하고,
85%가 입시 기준이 아닌 취향으로 자료를 선택해
판단 오류율이 65%에 달합니다.
```

**자막** (통계 강조)
```
132분 허비 | 65% 오류율
입시 기준 ❌ → 취향 선택 ✓
```

**연출 포인트**
- 통계 숫자: 애니메이션 (카운트업 효과)
- 빨간색(Red-500) 사용으로 문제 심각성 강조

---

#### [22~38초] 솔루션 — AI + 데이터

**화면 전환**
- 모바일 앱 UI 시연 (화면 녹화 또는 목업)
- 그림 업로드 → AI 분석 → 결과 표시 (3단계 플로우)

**앱 화면 시나리오**
```
1. [3초] 그림 업로드 화면 (버튼 탭 애니메이션)
2. [2초] "AI 분석 중..." 로딩 (프로그레스 바)
3. [5초] 5단계 평가 결과 + Top 10 큐레이션
```

**내레이션**
```
아트프렙은 비전 AI와 합격 DB 1만건 교차분석으로
이 132분을 10초로 단축하고,
합격 패턴 기반 큐레이션을 제공합니다.
```

**자막**
```
132분 → 10초 단축 🚀
합격 DB 1만건 교차분석 📊
```

**연출 포인트**
- 앱 화면: 실제 작동하는 것처럼 녹화 (Figma 프로토타입 활용 가능)
- 배경음악: 밝고 경쾌한 톤 시작
- 초록색(Emerald-500) 사용으로 긍정적 변화 강조

#### [38~50초] 트랙션 — 검증 완료

**화면 전환**

**그래픽 내용**

```
2026.01: 전환율 22% (업계 평균 2.2배) 🚀
2026.05: 로컬 프로토타입 완성 ⚙️
```

**내레이션**
```

```

**자막**
```

사전신청 125명 확보 ✅
```

**연출 포인트**
- 타임라인: 좌→우 와이프(Wipe) 애니메이션

## ✂️ 편집 가이드 (A안·B안 공통)

### 편집 소프트웨어
- **CapCut** (무료, 모바일·PC, 초보자 추천)
- **DaVinci Resolve** (무료, 전문가용, 고급 기능)
- **Premiere Pro** (유료, Adobe 구독 필요)
- **Final Cut Pro** (Mac 전용, 유료)

### 자막 스타일
- **폰트**: Pretendard Bold 또는 Noto Sans KR Bold
- **크기**: 32~40px (1080p 기준)
- **색상**: 화이트 (#FFFFFF) + 검은 테두리 (Stroke 2px) 또는 반투명 검은 박스 배경
- **위치**: 화면 하단 1/3 (얼굴 가리지 않도록)
- **애니메이션**: Fade In/Out (0.2초)

### 배경음악 믹싱
- **볼륨**: -20dB ~ -25dB (내레이션 방해 금지)
- **페이드 인/아웃**: 시작 2초 + 끝 2초
- **장르**: 밝고 희망적인 Corporate Music (너무 신나지 않게)

### 컬러 그레이딩
- **톤**: 밝고 깔끔 (대비 살짝 높이기)
- **채도**: 자연스럽게 (과한 색보정 지양)
- **룩**: Clean & Modern (필터 과용 금지)
