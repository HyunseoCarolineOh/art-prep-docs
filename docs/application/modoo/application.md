## 📋 Q2. 아이디어를 떠올린 배경 (필수, 1000자)

### 현장에서 발견한 진짜 문제

- **2년간 200명 넘는 미대 수험생 컨설팅** 진행
- 공부에 비해 미술 실력이 잘 늘지 않는 이유는, 노력이 부족해서가 아니라 **입시 미술의 평가 기준 자체가 매뉴얼로 존재하지 않기 때문.**
- 합격에 가까운 그림인지 판단하는 기준은 **수많은 합격작·채점 사례를 직접 본 강사의 '감'**으로만 존재 (객관화된 자료 부재)
- 학생은 합격 데이터 접근 자체가 차단됨 (학원이 사실상 독점·비공개) → '감'을 스스로 기를 방법 없음 → **모든 판단이 강사 1인에 종속**
- 그 결과 학생 **85%**가 평가 기준이 아닌 자기 취향으로 자료 선택, **수개월 잘못된 방향으로 연습 후 처음부터 재시작** 반복

### 외부 검증과 시장 니즈 확인

- 2023년 **홍익대 경영대학 아이디어 경진대회 1위** 수상 → 외부 시장성 검증 통과
- 2025년 **150명 온라인 설문 + 10명 심층 인터뷰** 실시
  - **78%**가 "탐색 시간 낭비·자기 그림 분석 부재"를 최우선 과제로 지목
  - **67%**가 월 3만원 이상 유료 결제 의향 → 가격 수용도 확인

### 창업을 결심한 계기

- **2024년 GPT-4 Vision·Gemini 등 비전 AI가 상업 API로 공개** → 그림 한 장을 정확히 분석할 수 있게 됨.
- 대표가 2년간 정리한 채점 기준을 AI 프롬프트로 코드화하면 강사의 '감'을 시스템으로 재현 가능
- 시장 니즈(확인) + 기술 실용화(2024) + 도메인 자산(2년 컨설팅) **3가지 요소를 갖추게 되어, 창업을 결심하게 됨.**

### MVP 시장 검증

- 2026년 1월 **MVP 랜딩페이지 출시** (서비스 가설 압축, 사전신청 수집)
- 방문자 **22%가 사전신청 전환** (업계 평균 5~10%의 **2.2배**)
- 광고 없이 **사전신청 125명** 확보 (자생적 입소문)
- 1:1 인터뷰: "드디어 이런 게 나왔다" 반응 다수
- 현재 본격 서비스 개발 + 합격 DB 구축 단계 진입

## 📋 Q3. 누구의 어떤 문제 (필수, 1000자)

### 대상

- **연 5만명 미대 수험생**
- 특히 **비수도권 거주 수험생** (수도권 거점 학원 접근 어려움)
- 특히 **학종·비실기 전형 준비생** (실기 학원 인프라가 닿지 않는 영역)

### 문제 1. 시간 낭비

- 일평균 **132분**을 인스타그램·핀터레스트·미대입시닷컴 등 SNS·커뮤니티 탐색에 허비
- 입시 특화 검색 도구가 없어 키워드를 바꿔가며 수백 장 무차별 스크롤
- 본 자료가 내 목표 학교·전형에 맞는지 판단할 객관적 기준 부재

### 문제 2. 판단 오류

- 학생 **85%**가 입시 평가 기준이 아닌 개인 취향으로 자료 선택 → 자료 판단 오류율 **65%**
- "내가 보기에 예쁘다"와 "이 그림이 실제 합격 패턴에 부합한다"는 완전히 다른 기준
- 이 둘을 구분해줄 도구가 시장에 부재 → 학생이 자기 판단이 옳은 방향인지 확신할 수 없음

### 문제 3. 정보·비용 격차

- 합격 데이터와 컨설팅은 수도권 거점 학원에 집중 → 지방 수험생은 정보 차단
- 지방 수험생은 수도권 학원 이용 시 **이동·거주비 이중 부담**
- 학원비 **월 200만원** 수준으로 가계 부담 큼 → 경제력 기준 진입장벽

### 시장 규모

- **TAM 1.2조원**: 국내 미술 사교육 시장 (통계청 기준)
- **SAM 약 2,400억원**: 미대 입시 사교육 시장
- **SOM 약 269억원**: 연 5만 수험생 × 월 29,900원 × 평균 18개월 구독 가정

> 시간·비용·판단을 동시에 낭비시키는 구조적 문제를 데이터와 AI로 해소.

**사이트 측정 글자수**: 692자 ✅

## 📋 Q4. 어떻게 실현 (필수, 1000자)

### 솔루션

- 학생이 자기 그림을 업로드하면, **구도·비례·명암·표현·완성도 5개 평가축으로 AI가 자동 분석한 리포트**를 제공
- 동시에 **합격작 DB 1만 건 중 평가축이 유사한 합격 그림을 자동 매칭**해 큐레이션 → "내 그림이 어디쯤 와 있고, 다음에 무엇을 봐야 하는지"를 객관적인 기준으로 제시

### 실행 로드맵

- **Phase 1 (2026 상반기)**: 모바일 웹앱 MVP 정식 출시(Next.js + Supabase), 사전신청 125명 중 파일럿 100명 확보, 반복 사용률 40%+ 검증
- **Phase 2 (2026 하반기)**: 합류한 데이터 라벨링·분석가 기반으로 합격 DB 2,000건 → 5,000건 확대, MAU 1,000명·유료전환율 25% 달성, 학원장 네트워크 20곳 기반 B2B 제휴 5곳
- **Phase 3 (2027)**: 합격 DB 1만 건 달성, B2B SaaS(학원용 인사이트 리포트) 정식 출시, 시드 1억원 유치

### 수익 모델

- B2C 프리미엄 구독: **월 29,900원** (학원비 월 200만원 대비 가계 부담 **90% 이상 절감**)
- B2B 학원용 리포트: **월 199,000원**
- **LTV/CAC 5.4** (SaaS 건강 기준 3.0 상회)

### 경쟁 우위

- 미대입시닷컴(커뮤니티 게시판), 인스타·핀터레스트(취향 검색) → 입시 기준 필터링·AI 분석 불가
- 경쟁사가 단기간에 따라오기 어려운 **3가지 핵심 자산** 결합:
  1. **분기형 프롬프트 트리**: 대표가 2년간 200명 컨설팅에서 정리한 채점 기준을 그림 유형에 따라 평가 흐름이 분기되는 AI 로직으로 코드화
  2. **합격작 DB 라벨링 체계**: 대학·전형·평가축별로 태깅된 합격 사례 데이터셋
  3. **행동 편차 메트릭**: 학생이 저장·열람한 자료가 합격 기준에서 얼마나 떨어졌는지 수치화 (개인 취향과 합격 기준 간극을 시각화)
- **3자산 모두 사용량·데이터·시간이 쌓일수록 두꺼워지는 복리형 해자**

## 🎯 면접 예상 질문 대비

### Q1. 비전 AI 모델 선정·운용은?
GPT-4 Vision·Gemini·Claude Vision 후보 벤치마킹 중. 프롬프트 트리가 모델 독립적이라 교체에 견고. 이미지당 ~13원, 월 1만건 분석 시 13만원으로 매출 대비 5% 미만.

### Q2. 합격 DB 1만 건 확보 + 저작권?
① 팀원 학원장 네트워크 20곳 협의 ② 컨설팅 출신 합격생 200명 직접 컨택 ③ 온라인 커뮤니티 제휴 ④ 데이터 라벨링 담당자 채용 완료. 약관·개별 계약(데이터 제공 대가 또는 무료 구독) + 개인정보 비식별화로 확보.

### Q3. 미대입시닷컴이 AI 따라오면?
미대입시닷컴은 게시판 커뮤니티 → AI 전환 시 프롬프트 트리(2년 채점 코드화), DB 라벨링 체계, 행동 데이터 수집 파이프라인 모두 처음부터 구축해야. 1~2년 데이터 비대칭이 진입장벽.

### Q4. 지방 수험생이 정말 결제할까?
2025년 설문(N=150): 결제 의향 67%, 지방 응답자 결제 의향 71%(수도권 64%보다 높음). 학원비 월 200만원 대비 구독료 월 29,900원이 90%+ 절감 + 이동·거주비 부담 제거.

### Q5. MAU 1,000명 달성 근거?
사전신청 125명 × 파일럿 전환 80% = 100명, 입소문·팀 네트워크 월 50명 × 6개월 = 300명, 광고비 월 50만원 × 6개월 = 600명. CAC ≈ 5,000원, LTV 538,200원 대비 LTV/CAC = 107.

### Q6. 기술 역량 충분?
대표가 AWS SAA + Supabase 백엔드 직접 구축 + Claude Code·Cursor 활용 프로토타입 완성. MVP는 자체 구현 가능. 본격 확장 시 CTO 채용.
