## Phase 1 — MVP

**기간**: 2026.04 ~ 2026.06 (9주, sprint 4/22~6/20)
**목표**: 핵심 가치 검증 + 파일럿 100명 운영

### 기능 범위

| 기능 | 상세 | 완료 기준 |
|------|------|-----------|
| 합격 DB 탐색/검색 | 실기유형 × 학교지역 필터 + 이미지 그리드 뷰 | 필터 3종 이상, 이미지 그리드 렌더링 |
| AI 스마트 큐레이션 | 프로필 입력 → 비전 AI 매칭 → Top10 추천 + **LightPrep A안 (5블록 AI 리포트, [05_LIGHTPREP_A_PRD](./05_LIGHTPREP_A_PRD.md))** | Top10 결과 + 추천 이유 표시 + 레퍼런스 1건당 5블록 리포트 생성 |
| 저장 기능 | 하트 버튼 + 저장 보드 | 저장/삭제 + 보드 페이지 |
| 기본 인증 | 이메일+비밀번호 (Supabase Auth) | 회원가입/로그인/로그아웃 |
| Vercel 배포 | 자동 CI/CD | 도메인 연결 + HTTPS |

### 성공 지표
- 파일럿 사용자 100명 등록
- 1인당 평균 세션 길이 5분 이상
- 저장 기능 사용률 60% 이상

### 기술 제약
- 결제 없음 (무료 제공)
- 이미지: Supabase Storage 무료 플랜 (1GB)
- AI 비용: 비전 AI API 종량제 (월 예산 확인 필요)

## Phase 2 — 확장

**기간**: 2026.06 ~ 2026.10 (약 4개월)
**목표**: 수익화 검증 + 유료 전환율 25%

### 기능 범위

| 기능 | 상세 |
|------|------|
| **5단계 AI 분석 리포트** | 사용자 작품 업로드 → 비전 AI → 5단계 JSON 리포트 (구도/명암/선/완성도/합격가능성) |
| **구독 결제 시스템** | 토스페이먼츠 or Stripe 연동, 월 29,900원 |
| **행동 데이터 개인화** | 저장/열람 패턴 분석 → 큐레이션 추천 보정 |
| **AnalysisReport 엔티티** | DB 스키마 확장 |

### 결정 필요 사항 (Phase 2 시작 전)
- [ ] 결제 시스템: 토스페이먼츠(국내) vs Stripe(글로벌) 결정
- [ ] 비전 AI 프롬프트 5단계 상세 설계
- [ ] 구독 플랜 가격 최종 확정

### 성공 지표
- 유료 전환율 25% 이상
- MAU 300명
- AI 리포트 사용 만족도 4.0/5.0 이상

## Phase 3 — 고도화

**기간**: 2026.10 ~ 2027.06 (약 8개월)
**목표**: B2B 확장 + MAU 1,000명

### 기능 범위

| 기능 | 상세 |
|------|------|
| **B2B 학원 대시보드** | 학생 관리 + 인사이트 리포트, 월 199,000원 |
| **소셜 로그인** | Google OAuth + Kakao 로그인 |
| **성능 최적화** | 이미지 CDN, 캐싱, 로딩 속도 개선 |
| **모바일 앱** | React Native or PWA 검토 |

### 성공 지표
- MAU 1,000명
- B2B 학원 10곳 이상 계약
- NPS 50 이상
