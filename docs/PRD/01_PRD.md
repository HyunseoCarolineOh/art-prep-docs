## 1. 제품 개요

### 한 줄 요약
연 5만명 미대입시생의 실기자료 탐색 132분을 10초로 단축하는 AI 큐레이션 플랫폼

### 해결 문제
| 문제 | 수치 |
|------|------|
| 비정형 SNS(인스타/핀터레스트) 탐색 시간 | 평균 **132분/회** |
| 참고작 품질 판단 오류율 | **65%** |
| 학교별/유형별 합격 기준 정보 파편화 | 신뢰 가능한 단일 출처 없음 |

### 핵심 가치 제안
**비전 AI 모델 × 합격 DB 1만건 교차분석 → 전략형 큐레이션**

- 검색 시간: 132분 → 10초
- 합격 DB 기반 신뢰성 있는 참고작 제공
- 목표 학교·유형에 맞는 맞춤 추천

## 2. 사용자

### 주요 타겟

| 세그먼트 | 특징 | 구독 플랜 |
|----------|------|-----------|
| **B2C: 미대입시생** | 10-20대, 수능 준비생, N수생 | 월 29,900원 (Phase 2) |

### Phase 1 타겟
- 미대입시생 B2C (파일럿 100명)
- 무료 제공 (결제 없음)

## 3. 핵심 기능

### Phase 1 MVP 기능

| 기능 | 설명 | 우선순위 | 복잡도 |
|------|------|----------|--------|
| 합격 DB 탐색/검색 | 실기유형 × 학교지역 필터 + 이미지 그리드 뷰 | P1 | 간단 |
| AI 스마트 큐레이션 | 프로필 입력 → 비전 AI 매칭 → Top10 추천 | P1 | 보통 |
| 저장 기능 | 하트 버튼 + 저장 보드 | P1 | 간단 |
| 기본 인증 | 이메일+비밀번호 (Supabase Auth) | P1 | 간단 |

### 기능 상세

#### 합격 DB 탐색/검색
- 필터: 실기유형(소묘/수채화/정물화/인물화 등), 학교, 지역, 합격연도
- 뷰: 이미지 그리드 (Pinterest 스타일)
- 정렬: 최신순, 관련도순
- Artwork 테이블은 **read-only** (관리자만 등록 가능)

#### AI 스마트 큐레이션
- 사용자 프로필 입력: 목표 학교, 실기유형, 현재 실력 수준
- 비전 AI API 호출 → 합격 DB 매칭 → Top10 추천
- 추천 이유 텍스트 포함
- 세션 저장 (CurationSession 테이블)

#### 저장 기능
- 작품 하트 버튼 (toggle)
- 저장 보드: 저장한 작품 모아보기
- SavedItem 테이블 연동

#### 기본 인증
- Supabase Auth (이메일+비밀번호)
- 회원가입, 로그인, 로그아웃
- 비회원: 탐색 가능, 저장/큐레이션은 로그인 필요
