# 합격하는 PSST 사업계획서 작성법
---

## 1. 창업 자금조달 개요

| 구분 | 사업화 지원사업 | 연구개발 지원사업 | 투자유치 |
|------|--------------|----------------|---------|
| **종류** | 창업패키지, 창중대, 소상공인, 지자체 등 | 디딤돌, 전략형, 산학연, R&BD 등 | Pre TIPS, TIPS, 민간투자 등 |
| **목적** | 사업화 지원 | 연구개발 지원 | 빠른 투자금 회수 / 최대 이익금 창출 |
| **평가** | 사업성 7 : 기술성 3 | 사업성 3 : 기술성 7 | 회수 가능성 |
| **필요 서류** | 사업계획서 (PSST) | 연구개발계획서 (PSST + 성능지표) | IR 자료 (PSST + Exit 전략) |
| **평가 요소** | 최종 산출물 + 성과지표 | 최종 산출물 + 성능지표 | 회수 가능성 |

> **핵심**: 자금조달의 시작과 기본기는 PSST 방식 사업계획서

---

## 2. 평가 트렌드 & 기본 원칙

### 대응 방향

**설명식 글 → 요약식 글(개조식)으로 전환**

- 불필요한 내용 작성 배제
- 누구나 이해하기 쉬운 워딩 사용
- 그림과 도표 최대한 활용

### 개조식 작성법

```
○ 큰 제목 1 (볼드 처리)
  - 하위 내용 A (1~3줄)
  - 하위 내용 B (1~3줄)
  - 하위 내용 C (1~3줄)
```

### MECE 법칙

> **상호배제(Mutually Exclusive) + 전체포괄(Collectively Exhaustive)**
> - 작은 내용끼리 중복이 없어야 함
> - 작은 내용의 합이 큰 내용과 일치해야 함

### 작성 순서 vs. 중요도

| | 순서 | 중요도 |
|--|------|--------|
| 1 | 본문 | 제목 |
| 2 | 요약 | 요약 |
| 3 | 제목 | 본문 |

## 3. PSST 프레임워크

```
PSST 방식 사업계획서
├── 신청·일반현황 + 요약
└── PSST 본문 + 기타 증빙서류
    ├── P - Problem  : "왜" 이 아이템이 필요한지
    ├── S - Solution : "어떻게" 개발해서 차별화할 것인지
    ├── S - Scale-up : "어떻게" 팔아서 성장해 나갈 것인지
    └── T - Team     : "누구"와 함께 실현할 것인지
```

## 4. Problem (문제)

**목적**: "왜" 이 아이템이 필요한지

### Problem-1 필수 요소

| 요소 | 설명 |
|------|------|
| 시장동향 및 증감추이 | 전체 시장의 증가 양상 |
| 문제점 (외적동기) | 시장/환경에서 발생하는 외부 문제 |
| 문제점 (내적동기) | 창업자 또는 소비자 내면의 동기 |

> **Tip**: 외적동기 2개 + 내적동기 1개로 구성

### Problem-2 필수 요소

| 요소 | 설명 |
|------|------|
| 소비자 Needs | 타깃 고객이 원하는 것 |
| 아이템 핵심기능 | 핵심기능 3개 → 소비자 Needs와 연계 |
| 정량적 기대효과 | 수치로 표현한 예상 효과 |

> **Tip**: 핵심기능 3개를 소비자 Needs와 명확히 연계하고, 기대효과는 반드시 정량적으로

---

### [예시] Problem - IT: "장전"

**Problem-1**

- **시장동향**: 서학개미 열풍으로 해외주식 보유 0.5조 → 30조 급증 / GDP 1% 규모 손실 발생
- **문제점(외적동기 1)**: 직관 80% 빈칙 투자로 손실 발생 — 개인 투자자의 정보 불균형
- **문제점(외적동기 2)**: 기존 해외 뉴스 서비스는 영문 원문 제공 수준에 그쳐 실시간 대응 불가
- **문제점(내적동기)**: 창업자 본인의 서학개미 경험에서 출발한 정보 격차 해소 욕구

**Problem-2**

- **소비자 Needs**: 실시간·번역·요약된 해외 주식 뉴스를 원스톱으로 받고 싶다
- **핵심기능**: ① AI 기반 뉴스 수집 ② 자동 분류·요약 ③ 한국어 번역 제공
- **기대효과**: 정보 접근 속도 60% 단축, 투자 의사결정 소요 시간 40% 감소

---

### [예시] Problem - 제조: "에코플라텍"

**Problem-1**

- **시장동향**: 연평균 3만t 이상 폐전선 발생, 매년 1,500억 원의 처분비용 소요
- **문제점(외적동기 1)**: 폐전선 내 금속은 회수하나 폐플라스틱(피복)은 대부분 소각·매립
- **문제점(외적동기 2)**: 소각 처리 시 환경오염(다이옥신 등) 문제 대두, 규제 강화 추세
- **문제점(내적동기)**: 환경 소재 창업 경험에서 출발한 폐자원 순환 욕구

**Problem-2**

- **소비자 Needs**: 품질 안정적인 재활용 플라스틱 소재를 저렴하게 공급받고 싶다
- **핵심기능**: ① 폐전선 피복 분리 기술 ② 고순도 펠릿 압출 성형 ③ 규격화된 B2B 공급
- **기대효과**: 처분비용 대비 소재 판매가로 수익 역전, 기업당 연 처리비용 30% 절감

## 5. Solution (솔루션)

**목적**: "어떻게" 개발해서 차별화할 것인지

### Solution-1 필수 요소

| 요소 | 설명 |
|------|------|
| 선행 준비현황 | 이미 한 것 (기술 검증, 프로토타입 등) |
| 전체 개발/제조 프로세스 | 한 것 → 할 것 명확히 구분한 흐름도 |
| 최종산출물 및 차별성 요소 | MVP 형태와 경쟁 대비 우위 포인트 |

> **Tip**: 한 것과 할 것을 명확히 구분, 구현 프로세스(레시피)는 반드시 이미지로 표현

### Solution-2 필수 요소

| 요소 | 설명 |
|------|------|
| 협약기간 내 사업계획 | 월별/분기별 마일스톤 |
| 사업비 집행계획 (1단계) | 초기 집행 항목·금액 |
| 사업비 집행계획 (2단계) | 후기 집행 항목·금액 |

> **Tip**: 협약기간 내 반드시 최종 산출물 MVP가 산출되도록 플랜 설계

### 예산 편성 Tip

- **지원금 Max로 신청** → 남으면 광고선전비로 채울 것
- **기관 승인 후 추후 변경 가능** → 처음부터 완벽할 필요 없음
- **Rough하게 작성 권장** → 세부 항목보다 큰 그림 우선
- **1~2인 인건비 편성 권장** → 대표자 인건비 포함

**선행 준비현황**

|------|------|----------|

     ↑                                                              ↓

|------|-----------|
| 앱 다운로드 수 | 93.4% |
| 월 활성 사용자 | 83.8% |
| 유료 전환율 | 86.8% |
| 파트너 체결 수 | 68.8% |

## 6. Scale-up (확장)

**목적**: "어떻게" 팔아서 성장해 나갈 것인지

### Scale-up-1 필수 요소

| 요소 | 설명 |
|------|------|
| 경쟁사 분석 및 상대우위 요소 | 경쟁사 대비 핵심 차별화 포인트 |
| 목표시장 TAM-SAM-SOM-LAM | 시장 규모를 단계별로 수치화 |
| 비즈니스 모델 (수익구조) | 누구에게 무엇을 팔아서 돈을 버는가 |

> **Tip**: 경쟁사 대비 우위요소를 근거로 목표시장 진입 및 수익화 가능성 어필

### Scale-up-2 필수 요소

| 요소 | 설명 |
|------|------|
| 사업화 전략 (마케팅 전략) | 초기 모객 Setting 중심 |
| 성과지표 달성전략 | Big 4 지표 → 목표 + 달성 근거 |
| 전체 사업계획 & ESG 실현계획 | 협약 후 전체 로드맵 |

> **Tip**: 마케팅 전략은 초기 모객 Setting 중심으로 / 성과지표 Big 4 → 목표 + 근거로 구성

### 성과지표 Big 4

- **매출액** (또는 매출계획)
- **고용창출** (인원수)
- **투자유치** (금액)
- **지식재산권** (특허, 상표 등)

---

### [예시] Scale-up - IT: "장전"

**경쟁사 분석**

| 구분 | JANGJEON | 구글 뉴스 | 블룸버그 | 연합인포맥스 |
|------|----------|---------|---------|------------|
| 타깃 고객 | 국내 개인투자자 | 일반 사용자 | 기관투자자 | 증권사·기관 |
| 서비스 형태 | App + Web | Web | Web | Web |
| 뉴스 번역·요약 | AI 자동 제공 | 없음 | 없음 | 없음 |
| 실시간 해외뉴스 | AI 글로벌 수집 | 크롤링 | 유료 구독 | 유료 구독 |
| **핵심 우위** | **수집-분류-요약-번역 AI 통합** | - | - | - |

**목표시장 (TAM-SAM-SOM-LAM)**

| 구분 | 설명 | 규모 |
|------|------|------|
| TAM | 국내 주식 투자자 전체 | 약 1,400만 명 |
| SAM | 해외주식(서학개미) 투자자 | 약 320만 명 |
| SOM | App 서비스 유료 전환 목표 | 약 3만 명 (1년차) |
| LAM | 동남아·개도국 글로벌 확장 | 장기 목표 |

**비즈니스 모델**

| 국가 | 소비자 형태 | 수요처 | 제공 서비스 | 비중 |
|------|-----------|-------|-----------|------|
| 국내 | B2B | 국내 증권사 | 트래픽 기반 광고, API 제공 | 22% |
| 국내 | B2B | 국내 자산운용사 | API 제공 | 9% |
| 국내 | B2C | 개인 투자자 | 월간 구독 서비스, 프로그램 매매 서비스 | 62% |
| 해외 | B2B | 해외 증권사 | API 제공 | 4% |
| 기타 | B2C | 해외 개인투자자 | 월간 구독 | 3% |

**마케팅 전략**

- 전략 1: 증권사 API 제공을 통한 MTS 교체 유입 마케팅
- 전략 2: SNS 채널을 활용하여 App 내 승인 기사 주기적 업로드
- 전략 3: 해외 주식투자 카페와 MOU를 통한 기사 및 콘텐츠 제공
- 전략 4: 프로그램 매매 등 서비스 고도화 → KPI 확보 → 투자유치를 통한 성장 가속
- 전략 5: 동남아 등 개도국 타깃으로 해외판 "장전" 개발로 글로벌 진출

## 7. Team (팀)

**목적**: "누구"와 함께 실현할 것인지

### 필수 요소

| 요소 | 설명 |
|------|------|
| 대표이력 및 보유역량 | 아이템과 유관한 경력 위주 |
| 팀원 현황 및 고용계획 | 현 팀원 + 협약기간 내 채용 계획 |
| 협력 기업 및 기관 | MOU, 파트너십, 멘토링 기관 등 |

## 8. 요약 및 제목

### 요약의 중요성

> 아이템을 이해하고 글을 보는 것과
> 글을 보면서 아이템을 이해하는 것은 **전혀 다르다**
> → 요약 파트가 매우 중요

### 요약 양식 (창업아이템 개요)

| 항목 | 기재 내용 |
|------|---------|
| **명칭** | 아이템 이름 + 법인주 (OS/플랫폼 형태) |
| **아이템 개요** | 제품·서비스 개요 (사용 용도, 사용자, 가격 등) / 핵심기능과 고객에게 제공하는 가치 |
| **배경 및 필요성** | 제품·서비스 개발의 필요성과 해결방안 / 목표시장(고객) 현황 및 요구사항 분석 |
| **현황 및 구체화 방안** | 사업 참여 이전 준비사항 / 협약기간 내 개발 예정인 최종 산출물 / 팀원·협력 현황 |
| **목표시장 및 사업화 전략** | 수익화 모델(BM) / 사업화 성장 전략 / 경쟁 대비 차별성·보유역량 |
| **이미지** | 아이템 특징을 나타내는 사진 또는 설계도 (2컷 이상) |

### 제목 작명 공식

> ① [타깃] 또는 [문제점]
> ② [핵심 기술] 또는 [핵심 기능]
> ③ [구현 채널] 또는 [구현 형태]
> ④ [아이템 명칭]

**IT 예시 - "장전"**:
```
[320만 서학개미를 위한] [AI 기반]의
[실시간해외뉴스 수집-분류-요약-번역]
[원스톱 App 솔루션] ["장전"]
```

**제조 예시 - "에코플라텍"**:
```
[연평균 3만t 이상 발생해]
[매년 1,500억 원의 처분비용이 소요되는]
[폐전선 내 폐플라스틱을 업사이클링한]
[펠릿] ["에코플라텍"]
```
