### **1.1.1. 3,900억 수험 시장의 구조적 전환 (시장 동향)**

▲ 국내 고등학생 미술 사교육비(3,942억원)는 예체능 전 과목 사교육비의 34.3%를 차지함.

- **경쟁 심화**: 주요 미술대학 입학 정원은 지속 감소(2024~2025년 추세)하는 반면 수험생 수는 유지·증가하여, 합격 데이터 기반 '전략적 학습'에 대한 수요가 구조적으로 급증하고 있음.
- **수요 전환**: 과거의 '감각 중심 학습'에서 '데이터 기반 전략 학습'으로 패러다임이 이동 중이며, 이 수요를 충족하는 플랫폼은 현재 시장에 존재하지 않음.

### **1.1.2. AI 전환점과 교육 격차가 만든 선점의 골든타임 (외적 동기)**

세 가지 외부 환경 변화가 맞물리며 지금이 ArtPrep 창업의 적기임을 보여준다.

**① 기술적 전환점 — 이미지 AI의 상업화**
Google DeepMind Gemini 2.0 Flash(비전 모델) 실용화(2023~)로 이미지 기반 실기자료 AI 분석이 처음으로 상업적으로 실현 가능해졌다. 선점 사업자가 데이터 플라이휠을 구축하면 1~2년 내 경쟁 복제가 구조적으로 어려워지므로, 지금이 시장 선점의 골든타임이다.

**② 사회적 불평등 — 수도권 집중의 구조화**
입시 정보와 합격 데이터가 수도권 특정 거점에 집중되어 있어, 지방 수험생은 이동·거주비 등 사회적 비용을 추가로 부담하거나 정보 접근 자체를 포기한다. 이는 교육 기회의 지역 불평등을 고착화하는 구조적 문제다.

**③ 시장 공백 — 기존 플랫폼의 구조적 한계**
현재 입시 플랫폼(미대입시닷컴, 인스타그램, 핀터레스트 등)은 커뮤니티·취향 기반 검색에 머물며, AI 분석·합격 DB 기반 전략형 큐레이션은 전무하다. 기술이 준비된 지금, 이 공백은 AI 기반 플랫폼이 진입할 수 있는 기회 영역이다.

### **1.1.3. 6년 현장이 증명한 구조적 반복 (내적 동기)**

6년간 미대입시 컨설팅 현장에서 200명 이상의 수험생을 관찰한 결과, 동일한 패턴이 반복됨을 확인했다. **학생의 85%가 입시 기준이 아닌 개인 취향으로 실기 자료를 선택하며, 이로 인해 방향성 정체 → 재설정 → 반복의 사이클에 빠진다.** 이는 개별 학생의 문제가 아니라, 올바른 판단을 도울 도구가 없는 구조적 부재에서 비롯된다.

이를 정량 검증하기 위해 2025년 11월 미대입시생 150명을 대상으로 온라인 설문을 실시한 결과:
- **78%**: '탐색 비효율과 분석 부재'를 최우선 해결 과제로 지목
- **67%**: 월 3만 원 이상 유료 결제 의향 표명

이와 같이 미술대학 입시가 지니는 구조적 문제를 현장 도메인 지식과 AI 기술을 결합하여 해결하고자 한다.

### 2.1.2. 솔루션 제품 플로우

*✓ **핵심 1**: **(분석)** AI는 개별 실기자료를 출제 의도 → 학교별 성향 → 조형 요소(구도·형태·색채·표현력) → 평가 기준 부합도 순으로 단계별 분석하여 실제 합격 데이터와 비교한 정량·정성 통합 리포트를 제공*

| 단계 | 사용자 상황 | AI 분석 | 사용자가 얻는 것 |
| --- | --- | --- | --- |
| 1단계 | 주제를 봤는데 어떤 방향인지 모름 | 출제 의도 파악 | 방향 설정 가이드 |
| 2단계 | 내 그림이 목표 학교와 맞는지 불안 | 학교별 성향 매칭 | 합격 패턴 비교 |
| 3단계 | 구도·색채·표현이 부족한 것 같음 | 조형 요소 분석 | 항목별 점수 |
| 4단계 | 기준에 맞게 그리고 있는지 불확실 | 평가 기준 부합도 | 통합 점수 |
| 5단계 | 합격 가능성을 알고 싶음 | 합격 데이터 비교 | 종합 진단 리포트 |

*✓ **핵심 2**: **(행동 데이터 기반 개인화 고도화)** 사용자의 저장, 반복 열람, 필터 선택 등의 행동 데이터를 실시간 수집하여 합격 데이터 구조와 비교 분석함. AI는 사용자의 취향 편향과 합격 기준 간 편차를 계산하여 추천 알고리즘을 지속적으로 보정하며, 사용 기간이 길어질수록 개인화 추천 정확도를 95% 이상 수준으로 고도화함.*

```plain text
행동 수집      →   편차 계산          →   알고리즘 보정       →   정확도 향상 (↩ 반복)
저장/열람/         취향 vs 합격기준       추천 가중치              95%+ 개인화
필터 행동          간 편차 측정           실시간 조정              수렴 달성
```

*✓ **핵심 3**: **(전략형 필터 시스템)** 실기유형과 학교 지역을 단위로 데이터를 분리하고, 각 유형별 합격작의 공통 구조를 분석하여 제공. 결과적으로 학생은 자신이 준비하는 정확한 시험 유형에 최적화된 자료만 선별하여 확인할 수 있음. 이는 불필요한 탐색 시간을 제거하고 실기 방향성을 초기에 교정하는 핵심이 됨.*

```plain text
실기유형 × 학교지역 필터 체계
├── 실기유형: 소묘 / 수채화 / 유화 / 기초디자인 / 발상과표현 / 사고의전환 / 개체묘사 / 기초소양 / 입체조형 외
└── 학교지역: 수도권(서울/경기) / 충청권 / 호남권 / 영남권
```

### 단위 경제학 (Unit Economics)

아트프렙의 수익 구조는 구독형 SaaS 모델의 핵심 지표인 LTV/CAC 비율에서 강력한 경제성을 갖는다.

| 지표 | 산출 근거 | 값 |
| --- | --- | --- |
| **LTV (고객 생애가치)** | 월 29,900원 × 12개월 × 평균 구독 기간 1.5년 | **538,200원** |
