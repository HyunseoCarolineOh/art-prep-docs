## 2. 창업 아이템 개요(요약)

| 항목 | 내용 |
|------|------|
| **명칭** | 아트프렙(ArtPrep) |
| **범주** | AI 기반 교육 플랫폼(EdTech SaaS) |
| **아이템 개요** | 비전 AI 모델 기반 합격 DB 1만건 교차분석 → 10초 내 전략형 큐레이션 + 5단계 AI 분석 리포트 제공. B2C 월 29,900원 구독 / B2B 학원 월 199,000원 인사이트 리포트 |
| **문제 인식(Problem)** | 연 5만명 수험생이 일평균 132분을 비정형 SNS 탐색에 허비, 판단 오류율 65%. AI 분석·합격 DB 기반 전략형 큐레이션 플랫폼은 현재 시장에 전무 |
| **실현 가능성(Solution)** | MVP 랜딩페이지 전환율 22%(125명 사전신청, 업계 평균 5~10% 대비 2.2배) 검증 완료. 협약기간 내 웹앱 1개 개발 및 특허·상표·디자인 3건 출원 목표 |
| **성장전략(Scale-up)** | SAM 5,000억~7,000억원 국내 미대입시 시장 공략, 26년 0.4억 → 29년 10억+ 매출 목표. LTV/CAC 5.4 기반 B2C 구독·B2B 리포트 이중 수익 구조 |
| **팀 구성(Team)** | 도메인 전문성·제품 기획·직접 개발 3축 역량 보유 대표자 + 영업·UX 팀원 2명(구성 완료). 데이터분석가·라벨링 담당자 26년 단계별 채용 예정 |
