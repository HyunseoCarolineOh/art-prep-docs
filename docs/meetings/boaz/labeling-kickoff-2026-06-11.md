**멘트**: "전처리 가이드 확인했다. 오늘은 작업 기준에서 애매한 부분, 규모, 6/19 일정의 현실성을 확정하고 싶다. 안건 4개."

**안건 목록 (화면 공유)**

2. 확인 질문: 규모·문구 기준·도구·산출물 위치

## 2. 작업 정의 리뷰 (10분)

**멘트**: "가이드를 이렇게 이해했다. 맞는지 확인 부탁."

**자료**: [boaz-preprocessing-guide.md](./boaz-preprocessing-guide.md) 화면 공유

**인라인 요약 (구두 확인용)**

| Task | 작업 | 산출 |
|------|------|------|
| 1 | 잘못 크롭된 사진 재크롭 (원본 폴더에서 찾아 덮어쓰기) | 정리된 `크롭된 사진` 폴더 |
| 2 | 중복 크롭 제거 (`(1)(1)`, `(1)(2)` 류 통일) | 〃 |
| 3 | 미완성작 제거 (전공자 판단: vision encoding에 충분한가) | 〃 |
| 4 | 작품당 한 줄 특성 문구 작성 | `특성 문구 라벨링.xlsx` (작품명·문구 2컬럼) |

**핵심 룰 재확인**: 합격작 아닌 것 같아도 임의 삭제 금지, 합격작 여부는 모델링 담당자 추후 처리

## 3. 확인 질문 → 확정 답변 (2026-06-11 정리)

| # | 질문 | 답변 | BOAZ 노티 |
|---|------|------|----------|
| Q1 | `크롭된 사진` 폴더 정확한 장수 | **2,982장 · 고유 작품 2,486개** (전부 jpg, 원본 2,811장). 파일명 기준 중복 클러스터 377개(496장 정리 가능). 원본 없는 크롭 2장, 크롭 안 된 원본 326장 (로컬 전수 분석, 6/11) | 공유 |
| Q2 | 특성 문구 작성 기준 | 가이드 예시 구조로 진행: "소재 + 표현 의도 + 실기유형" 한 문장 | 파일럿에서 검증 |
| Q3 | 문구 전수 vs 샘플 | 보존 작품 전체 전수 | 불필요 |
| Q4 | 크롭 도구·포맷 요건 | 자율 (지정 없음). 파일명·덮어쓰기 규칙만 준수 | 불필요 |

| Q6 | xlsx 업로드 위치 | 공유받은 Drive 폴더 루트 | 업로드 시 노티 |
| Q7 | 파일럿 검수 | 크롭 검수 100장 + 문구 20개 먼저 작업 → 천시원 피드백 → 전체 진행 | **필요** (검수 요청) |

## 4. 사전 작업·파일럿 결과 공유 (10분)

**멘트**: "착수 전에 전체 인벤토리 분석과 파일럿 검수를 AI로 먼저 돌려봤다. 결과 공유하고 5가지만 확인받고 싶다."

**완료된 사전 작업 (6/11 기준)**

| 작업 | 결과 |
|------|------|
| 전수 인벤토리 | 크롭 2,982장(고유 작품 2,486개) · 원본 2,811장, CSV 5종 |
| 파일명 중복 클러스터 분석 | 377개 클러스터, 정리 시 496장 감소 |
| 파일럿: 클러스터 20개 시각 판정 | 19개 same(1장만 남기면 됨), 1개 mixed |
| 파일럿: 크롭 검수 100장 | ok 34 · 재크롭 필요 57(워터마크·로고 36, 혼입 15, 잘림 5, 과정샷 1) · 미완성 의심 9 |
| 파일럿: 특성 문구 초안 20개 | 오현서 톤 승인 완료. 예: "광택 있는 빨간 사과와 집게, 파란 쓰레받기를 사선 구도로 묘사한 기초디자인 작품" |

**BOAZ 확인 요청 5건**

| # | 항목 | 우리 입장 |
|---|------|----------|
| 1 | 작품 위에 겹쳐 인쇄된 워터마크·로고 (빈도 1위, 크롭으로 제거 불가) | **보존**으로 진행 예정, 모델 학습 관점 이견 있으면 회신 |
| 2 | 크롭 안 된 원본 326장 | 크롭 누락인지 의도적 제외인지 확인 |
| 3 | 원본 없는 크롭 2장 | 재크롭 불가, 현상 유지 예정 |
| 4 | Drive 2,983 vs 로컬 zip 2,982 (1장 차이) | 동일 파일명 충돌 추정, 무시해도 되는지 |
| 5 | 특성 문구 샘플 20개 | 모델 학습 입력 관점에서 형식 컨펌 요청 |

## 5. 일정·분담 계획 (15분)

**멘트**: "3인(현서·혜진·정민) + AI 보조로 6/19까지 완료한다. 분담과 일정 공유."

**인원 분담**

| 담당 | 역할 |
|------|------|
| 한혜진·우정민 | 크롭 검수(플래깅 후보 확인) + 재크롭 실행 |
| 오현서 | 완성도 최종 판단(전공자) + 특성 문구 초안 검수 + 전체 총괄 |
| Claude (AI) | 아래 자동화 표 전담 |

**AI(Claude) 분담 — 사람 작업량을 줄이는 자동화**

| Task | Claude가 하는 것 | 사람이 하는 것 |
|------|-----------------|---------------|
| 사전 준비 | Drive 폴더 전수 카운트·파일명 인벤토리, 로컬 작업 폴더 구성 | Drive에서 폴더 zip 일괄 다운로드 (개별 API 2,500회보다 빠름) |
| Task 1 재크롭 | 이미지 배치 열람으로 잘못 크롭 후보 플래깅 (다른 그림 혼입·외부 글자·잘림·과정샷), 후보 리스트 시트 생성 | 혜진·정민: 플래깅 후보 확인 + 재크롭 실행 |
| Task 2 중복 | 파일명 패턴 `(1)(N)` 중복 자동 탐지 + perceptual hash(Python)로 내용 기반 중복 클러스터링 → 제거 후보 리스트 | 오현서: 클러스터별 대표 1장 승인 |
| Task 3 미완성 | 비전으로 명백 비합격작(로고·이모지·아이콘·백지 수준) 사전 스크리닝 → 의심 리스트 | 오현서: 완성도 최종 판단 (vision encoding 충분성, 전공자 몫) |
| Task 4 문구 | 작품별 특성 문구 초안 일괄 생성 ("소재 + 표현 의도 + 실기유형" 구조) | 오현서: 초안 검수·수정 |
| 마감 | `특성 문구 라벨링.xlsx` 조립, Drive 업로드, 진행률 집계, comm-log 기록·그룹방 메시지 초안 | 오현서: 최종 확인 후 전달 |

**일정 (6/19 마감 기준, 3인 + AI)**

| 날짜 | 작업 | 담당 |
|------|------|------|
| 6/11 (수) | ✅ 완료: zip 확보·인벤토리·중복 클러스터 분석·파일럿(클러스터 20 + 검수 100 + 문구 20, 톤 승인) · 킥오프 회의 | 오현서 + Claude |
| 6/12 (목) | 회의 피드백 반영 → **AI 전수 실행**: 중복 정리(same 클러스터 제거대상 이동) → 크롭 검수 약 2,500장(겹침 워터마크 보존 룰) → 문구 초안 전수 생성 | Claude |
| 6/13~16 (금~월) | AI 플래깅 리스트 기반 재크롭 · mixed 클러스터·미완성 최종 판정 | 혜진·정민 / 오현서 |
| 6/15 (일) | 중간 체크인: 진행률 그룹방 공유 | 오현서 |
| 6/17 (화) | 문구 초안 전수 검수·수정 | 오현서 |
| 6/18 (수) | 정합성 점검 (폴더 파일명 ↔ xlsx 매칭, 제거분 반영) + xlsx 조립 | 오현서 + Claude |
| 6/19 (목) | **Drive 업로드 · 천시원 노티 (Open #9 완료)** | 오현서 |
