## Step 1 — 인생 대목표 Recap

**미국에서 창업/사업 성공** (3-5년 지평)

- 단순 취업이나 이민이 아니라 **내 사업을 미국에서 일으키는 것**이 궁극 목표
- 세일즈 배경 위에 기술·프로덕트 레이어를 쌓아서 **창업자 정체성**을 완성
- 3-5년은 스타트업 궤도에 올리는 데 필요한 최소 시간

---

## Step 2 — 대목표를 이루기 위해 필요한 것들

본인이 "지금 가장 부족하다"고 꼽은 요소:

1. **기술/AI 프로덕트 빌딩 실력**
2. **End-to-end 스타트업 운영 트랙레코드**
3. **미국 시장/투자자 네트워크**
4. **영어 커뮤니케이션/피칭**
5. **자본/자금력** (투자 유치 경험 포함)
6. **바이럴성 있는 개인 브랜드/포트폴리오**

### 9주 안에 실제로 움직일 수 있는 것

| 요소 | 9주 내 레버 | 이번 스프린트에서 겨냥 |
|------|------------|-------------------------|
| 1. 기술/AI 빌딩 | 부분 가능 | Claude Code 오케스트레이션, 데이터팀 협업 간접경험 |
| 2. E2E 운영 트랙 | **가능** | **핵심 타겟** |
| 3. 미국 네트워크 | 어려움 | 패스 |
| 4. 영어 피칭 | 부분 가능 | 영어 case study 1편 (피칭 초안 씨앗) |
| 5. 자본 | 어려움 | 패스 |
| 6. 개인 브랜드/포폴 | **가능** | **핵심 타겟** (블로그·LinkedIn·배포된 서비스) |

3·5는 이번 사이클에서 의미있게 움직이기 어려움 → **다음 사이클**로 이월.

## Step 3 — 이번 프로젝트에서 얻고 싶은 것

우선순위 3가지:

1. **E2E 스타트업 운영 트랙레코드** — 기획→개발→GTM→인터뷰→의사결정 전주기 1회 완주 증명
2. **CEO/GTM 드라이브 경험** — "내 손으로 파일럿 100명 책임졌다"는 실전 경험
3. **포트폴리오 자산** — 블로그 2, LinkedIn 1, 영어 case study 1, 배포된 서비스 URL

이 3가지는 미국 창업이든 미국 취업이든 **둘 다에 효용 있음** (병행 전략).

## Step 4 — 스프린트 목표 (SMART)

**시작 4/22(수) · 공식 마감 6/20(토) · 내부 마감 6/16(화, W8 말)** — "1회 이상 작품 저장/탐색한 액티브 유저 100명" + 인터뷰 10건 + 포트폴리오 자산 3건. 스프린트 기간 8주+3일. W9(6/17-6/20, 4일)는 버퍼·최종 정리.

- **Specific**: 액티브 유저(가입만 아님)·인터뷰·포트폴리오 자산으로 측정 항목 명시
- **Measurable**: 대시보드에서 숫자 확인 가능
- **Achievable**: 학원 제휴 + 커뮤니티 노출 조합으로 현실적
- **Relevant**: Step 3의 3가지 목표 모두에 직접 연결
- **Time-bound**: 2026-06-20

## Step 5 — 가장 빠르게, 적은 리소스로 달성하는 법

### 현실 제약

- 본인: 풀타임 직장인, **주당 5-10시간** → 9주 × 7.5h ≈ **65h 총량**
- 병행 트랙: **미국 취업/비자 트랙이 최우선** (충돌 시 art-prep 양보)

### 전략 (5줄 요약)

1. **중심축 1개만**: GTM·파일럿 모집 드라이브 + 제품 PM + 유저 인터뷰
2. **위임 철저**: 데이터·AI·학원 제휴·디자인은 각 담당자에게
3. **Concierge 허용**: AI 모델 v0.1 전까지 분석 리포트는 수동 가능
4. **W3부터 베타 모집 병행**: 완벽한 MVP 기다리지 않음
5. **W5 pivot 게이트**: 알파 리텐션 부진하면 타겟·형태 변경

### 위임 구조

| 영역 | 담당 | 본인 관여도 |
|------|------|-------------|
| 데이터 파이프라인 + 대시보드 + AI 모델 | **데이터 팀** (본업 동료/지인) | R&R·지표 정의만 |
| 프론트 개발 | Claude Code (본인 오케스트레이션) | UX 의사결정 |
| 디자인 | 우정민 | 방향 승인 |
| 학원 제휴 네트워크 | **한혜진** | 피칭 덱 제공 |
| 마케팅·콘텐츠 | 한혜진 | 공동 드라이브 |
| 유저 인터뷰 | **본인** | 최고 관여도 |

### 리소스 배분 (65h)

| 영역 | 시간 |
|------|------|
| GTM·파일럿 모집 공동 드라이브 | 25h |
| 유저 인터뷰 10건 | 10h |
| 데이터 팀 R&R·싱크·방향 | 10h |
| 제품 UX 의사결정·온보딩 기획 | 8h |
| 포트폴리오 자산 (블로그·LinkedIn·영어) | 7h |
| 주간 회고·계획 | 5h |

### 게이트

- **W5 (5/22) 1차 게이트**: 알파 5명 중 ≥3명 주 1회 재방문 + 학원 1곳 반응 → 미달이면 타겟 재설계
- **W9 (6/20) 최종 게이트**:
  - `continue` = 액티브 100명 + 재방문율 ≥20% + 유료 의향 3명+
  - `pivot` = 숫자 애매하지만 특정 세그먼트 강한 반응
  - `kill` = 100명 미달 + 재방문율 <10% → 미국 트랙 올인

## 병행 트랙 관계

**art-prep이 실패해도 남는 자산**

- E2E 운영 경험 (이력서·면접 얘깃거리)
- 배포된 서비스 URL (포폴 링크)
- 블로그·LinkedIn·영어 case study
- 데이터 팀과의 협업 경험 (본업 관계 강화)

**art-prep이 성공하면 플러스**

- 한국 파일럿 → 미국 시장 확장 스토리
- 실제 유료 전환 고객 (피칭 근거)
- 팀 운영 증명

**시간 충돌 시 우선순위**: 미국 트랙 > art-prep.
