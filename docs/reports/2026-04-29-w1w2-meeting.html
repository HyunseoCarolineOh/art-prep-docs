01
오늘 무엇이 달라졌나

"화면만 있던 것" → "진짜로 데이터가 흐르는 것"

필터 / 저장 / 업로드 / 내 작품 — 네 영역이 같은 날 한꺼번에 살아났어요. 왼쪽이 어제까지, 오른쪽이 오늘부터.

1
어제까지

필터 옵션을 코드 안에 박아둠

"홍대" "서울대" 같은 항목을 추가하려면 매번 개발자가 코드를 고쳐야 했어요.

→
오늘부터

DB에서 항목 추가·삭제

운영자가 코드 손 안 대고 옵션을 바꿀 수 있어요. "미지정"은 항상 맨 뒤로 자동 정렬.

2
어제까지

하트 = 가짜 카운터

새로고침하면 사라지는 화면용 숫자였어요. "저장한 작품" 목록이 만들어질 수 없는 상태.

→
오늘부터

진짜 저장 + /saved 페이지

하트를 누르면 DB에 저장되고, 새로고침해도 살아있어요. 빠르게 두 번 눌러도 안전하게 처리.

3
어제까지

업로드 = 가짜 알림창

버튼을 누르면 "백엔드 연동 후 완성됩니다" 알림창만 떴어요. 사진이 어디로도 가지 않음.

→
오늘부터

진짜 업로드 + 자동 정리

사진이 실제로 올라가고 DB에 저장돼요. 도중에 실패하면 올라간 사진을 자동으로 지워서 쓰레기 파일이 안 남게 했어요.

4
어제까지

내가 올린 작품을 볼 곳이 없음

업로드를 해도 다시 찾아볼 수 없었어요.

→
오늘부터

"내 업로드" 페이지 신설

본인이 올린 작품만 모아 보고, 한 자리에서 삭제할 수 있어요. 삭제하면 사진 파일도 같이 정리.
