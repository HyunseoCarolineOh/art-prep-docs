# **1. 문제 인식 (Problem)**

## **1.1. 창업아이템의 배경 및 필요성**

### **1.1.1. 연 3,900억 규모 시장의 정보 비대칭 심화 (외적 동기)**

> *"인스타그램을 2시간 탐색한 뒤 취향에 맞는 참고작을 골랐지만, 불합격 통보를 받고서야 그것이 입시 기준과 무관한 선택이었음을 깨닫는다. 이 반복되는 오판의 비용은 단순한 시간 낭비가 아니라, 수험생 한 명의 1년이다."*

- **[시장 규모 및 변화]**: 국내 고등 미술 사교육 시장은 연간 3,942억 원 규모(예체능 전체의 34.3%)(출처: 서울시 교육청 사교육비 통계, 2024)로, 실기 반영률이 90%에 육박함(출처: 어디가 adiga.kr, 2024년 각 대학 모집요강)에 따라 '데이터 해석 역량'이 합격의 핵심 지표로 부상함.

▲ 국내 고등학생 미술 사교육비(3,942억원)는 예체능 전 과목 사교육비의 34.3%를 차지합니다.

```
▶ 예체능 사교육비 구성 (서울시 교육청, 2024)
미술 : ████████████████░░░░░░░░░░░░░░░░ 34.3% (3,942억원)
음악 : ████████████░░░░░░░░░░░░░░░░░░░░ 27.1%
기타 : ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░ 38.6%
```

- **[데이터 독점 구조]**: 합격작 데이터의 90% 이상이 소수 대형 학원의 내부 자산으로 사유화되어, 외부 수험생들이 전략 수립 단계부터 소외되는 구조적 모순 발생.
- **[지역 간 교육 격차]**: 정보가 수도권 특정 거점에 집중되어 지방 수험생의 이동·거주비 등 사회적 비용 발생 및 지역에 따른 교육 기회 불평등 야기.
- **[핵심 페인포인트]**: 공신력 있는 플랫폼 부재로 SNS(인스타 등) 비정형 데이터에 의존. 일평균 132분(연습 시간의 50%) (자체 조사, n=150)을 자료 탐색에 허비하며, 정답에 해당하는 자료가 아닌 개인적 취향에 따른 자료 선택으로 판단 오류율이 65%에 달함.

### **1.1.2. 6년간의 현장 경험과 MVP 컨셉 검증 완료 (실증적 동기)**

- **(1) [도메인 전문성]** 6년간 미대입시 강사 활동을 통해 200명 이상의 데이터 분석 결과, 학생 85%가 입시 기준이 아닌 '개인 취향'으로 자료를 선택해 성적이 정체되는 현상을 반복 확인.
- **(2) [시장 수요 검증]** 미대입시생 150명 대상 설문(온라인 구글폼, 2025년 11월, n=150) 결과, 78%가 '탐색 비효율과 분석 부재'를 최우선 과제로 지목. 응답자의 67%가 월 3만 원 이상의 유료 결제 의사 표명.
- **(3) [MVP 컨셉 검증]** AI 분석 리포트 랜딩페이지 운영 결과, 방문자 500명 중 110명이 사전 알림 신청(전환율 22%). 이후 직접 소통을 통해 현재 125명 확보. 업계 평균(5~10%, 랜딩페이지 업계 평균 전환율 기준 — 마케팅 업계 조사, 2024) 대비 2.2배 높은 시장 반응 확보.

## **1.3. 창업 적기 근거 (긴급성)**

- **기술적 전환점**: OpenAI GPT-4V(비전 모델) 실용화(2023~)로 이미지 기반 실기자료 AI 분석이 처음으로 상업적으로 실현 가능해짐. 선점자가 데이터 플라이휠을 구축하면 1~2년 내 경쟁 복제가 구조적으로 불가능해지므로, 지금이 시장 선점의 골든타임임.
- **구조적 수요 증가**: 미술대학 정원은 지속 감소 추세(2024~2025년 주요 미술대학 입학 정원 감소, 각 대학 2025 모집요강)인 반면 수험생 경쟁 강도는 높아져, 합격 데이터 기반 '전략적 학습' 수요가 급증하는 구조.
- **시장 공백**: 기존 입시 플랫폼이 커뮤니티·폐쇄형 학원 시스템에 머물러 있는 현 시점이 AI 기반 개방형 플랫폼의 선점 기회임.

> *이상의 문제 인식을 바탕으로, 다음 섹션에서는 이를 해결하는 기술적 실현 가능성과 구체적 개발 계획을 제시한다.*

### 2.1.2. 솔루션 제품 플로우

> *✓ **핵심 1**: 사용자가 목표 대학·전형·실기 유형을 선택하면, AI는 1만 건 이상의 합격 DB를 교차 분석하여 10초 이내 최적 합격작 및 참고작 100선을 우선순위 기반으로 추천함. 합격 기여도 점수를 반영한 전략형 큐레이션 시스템임.*

```
[사용자 입력]      →    [AI 교차 분석]        →    [전략형 추천 100선]
목표대학·전형·         1만건 합격DB               합격기여도 점수 기반
실기유형 선택          교차분석 (10초)             우선순위 자동 정렬
```

*✓ **핵심 2**: **(분석)** AI는 개별 실기자료를 출제 의도 → 학교별 성향 → 조형 요소(구도·형태·색채·표현력) → 평가 기준 부합도 순으로 단계별 분석하여 실제 합격 데이터와 비교한 정량·정성 통합 리포트를 제공*

| 분석 단계 | 분석 내용 | 출력 결과 |
|----------|----------|----------|
| 1단계 | 출제 의도 파악 | 주제 해석 가이드 |
| 2단계 | 학교별 성향 매칭 | 학교 합격 패턴 비교 |
| 3단계 | 조형 요소 분석 | 구도·형태·색채·표현력 점수 |
| 4단계 | 평가 기준 부합도 | 정량·정성 통합 점수 |
| 5단계 | 통합 리포트 | 합격 데이터 대비 종합 진단 |

*✓ **핵심 3**: **(행동 데이터 기반 개인화 고도화)** 사용자의 저장, 반복 열람, 필터 선택 등의 행동 데이터를 실시간 수집하여 합격 데이터 구조와 비교 분석함. AI는 사용자의 취향 편향과 합격 기준 간 편차를 계산하여 추천 알고리즘을 지속적으로 보정하며, 사용 기간이 길어질수록 개인화 추천 정확도를 95% 이상 수준으로 고도화함.*

```
행동 수집      →   편차 계산          →   알고리즘 보정       →   정확도 향상 (↩ 반복)
저장/열람/         취향 vs 합격기준       추천 가중치              95%+ 개인화
필터 행동          간 편차 측정           실시간 조정              수렴 달성
```

*✓ **핵심 4**: **(전략형 필터 시스템)** 실기유형과 학교 지역을 단위로 데이터를 분리하고, 각 유형별 합격작의 공통 구조를 분석하여 제공. 결과적으로 학생은 자신이 준비하는 정확한 시험 유형에 최적화된 자료만 선별하여 확인할 수 있음. 이는 불필요한 탐색 시간을 제거하고 실기 방향성을 초기에 교정하는 핵심이 됨.*

```
실기유형 × 학교지역 필터 체계
├── 실기유형: 소묘 / 수채화 / 기초디자인 / 사고의전환
└── 학교지역: 수도권(서울/경기) / 충청권 / 호남권 / 영남권
```

## 3.3. 사업성 및 수익 전망

### 단위 경제학 (Unit Economics)

아트프렙의 수익 구조는 구독형 SaaS 모델의 핵심 지표인 LTV/CAC 비율에서 강력한 경제성을 갖는다.

| 지표 | 산출 근거 | 값 |
|---|---|---|
| **LTV (고객 생애가치)** | 월 29,900원 × 12개월 × 평균 구독 기간 1.5년 | **538,200원** |
| **CAC (고객 획득 비용)** | 마케팅비 총액 ÷ 신규 유료 고객 수 (목표) | **< 100,000원** |
| **LTV / CAC** | 538,200 ÷ 100,000 | **≈ 5.4** |

> **LTV/CAC 5.4는 SaaS 업계 건강한 수익 구조 기준(3.0 이상)을 크게 상회한다.** 미대입시생은 수험 기간(평균 1~2년) 동안 높은 유지율이 기대되며, B2B 학원 채널을 통한 교차 판매로 CAC를 추가 절감할 수 있다.
