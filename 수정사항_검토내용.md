## 🔴 심각한 오류 (1순위 수정 완료)

### 1. TAM > SAM > SOM 역전 오류 ✅ [수정 완료]

**문제**: SOM(5,000억~7,000억원) > SAM(850억~1,500억원) 역전
- TAM → SAM → SOM 계층 관계 위반

**해결책**:
- **SAM** (유효 시장): "국내 온라인 미술 교육 시장" → **"국내 미대입시 실기 교육 시장 전체(오프라인 포함)"** 로 재정의 → 5,000억~7,000억원
- **SOM** (획득 가능): "국내 미대입시 실기 교육 시장" → **"국내 미대입시 실기 교육 시장 중 디지털/앱 기반 온라인 세그먼트"** 로 축소 → 850억~1,500억원

**위치**: 사업계획서 3.1.1 목표 시장 규모 분석 표

---

### 2. 개발자 채용 시기 vs 앱 개발 일정 불일치 ✅ [수정 완료]

**문제**: 개발자 채용 예정 '26.09 vs MVP 개발 일정 26.02~26.04 → "개발자 없이 어떻게 앱 개발?"

**해결책**:
- 1단계(26.02~26.04): **외주 개발사를 통한 MVP 개발** 명시
- 2단계(26.06~12): 내부 AI 개발자 채용(26.09) → 분석 엔진 고도화 및 유지보수 담당
- 개발 전략 설명 추가: 초기 외주 개발 → 파일럿 검증 → 내부 개발로 전환

**위치**:
- 사업계획서 2.4.1 (MVP 개발 및 데이터 구축 행 수정)
- 2.4.2 (1단계 정부지원사업비 설명 추가)
- 2.4.3 (2단계 정부지원사업비 설명 추가)
- 주석 [⁵] 추가

### 7. 26년 매출 계산의 모호성 ✅ [수정 완료]

**문제**: 연간 기준 기대 매출(0.64억)과 26년 목표(0.4억) 간 설명 부족

**해결책**:
- 26년 실제 매출 = 상반기(0.08억) + 하반기(0.32억) = 0.4억원
- 이는 **연간 기준이 아닌 운영 기간(상반기 파일럿, 하반기 본격 출시) 기반 산정**임을 명시

**추가 설명**:
```
26년 상반기(26.01~26.06): MVP 개발, 파일럿 진행 → 매출 0.08억원
26년 하반기(26.07~26.12): 정식 앱 출시 및 B2B 서비스 시작 → 매출 0.32억원
합계: 0.40억원
```

**수정 위치**:
- B2B 채널 요약 표 이후 "참고" 섹션 추가

#### 🔴 P1 (긴급 개선) - 경쟁사 분석 구체화

**위치**: 2.2.1 경쟁사 비교 테이블

**개선 내용**:
- **익명 제거**: "경쟁사 A/B/C" → "커뮤니티형/강사주관형/추천알고리즘형"으로 특성 명확화
- **경쟁사 한계점 추가**: 각 유형별 근본적 한계점 명시
  - 커뮤니티형: "검증 불가, 주관적 선택" → 사용자 판단 오류율 65%
